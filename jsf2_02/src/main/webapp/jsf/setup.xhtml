Set Up Your IDE and Project for JSF 2.2

  • Once only: register JSF 2.2 library. First, create JSF 2.2 user library via Window --> Preferences --> Java --> Build Path --> User Libraries --> New. Second, select that new library and point at JSF 2.2 JAR file. Do this by clicking on your new library, clicking "Add External JARs", and pointing to whereever you saved the JSF 2.2 JAR file.
  • Each time: make Dynamic Web Project with JSF 2.2 facet. First, File --> New --> Project --> Web --> Dynamic Web Project. Second, specify "JavaServer Faces v2.2 Project" for the Configuration.

For details, see the "Installation, Setup, and Getting Started" section of the JSF 2.2 tutorial.
